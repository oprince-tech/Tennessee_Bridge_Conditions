# Tennessee_Bridge_Conditions

### Installation:
```
git clone https://github.com/oprince-tech/Tennessee_Bridge_Conditions.git
cd Tennessee_Bridge_Conditions
. venv/bin/activate
pip install -r requirements.txt
```

### Usage:
```
python3 main.py
```
